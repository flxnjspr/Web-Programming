HTML & CSS Quick Reference Guide

A beginner's guide to basic HTML and CSS tags and properties with examples.

HTML Tags

<h1> - Heading 1

The <h1> tag is used to define the main heading. It's the most important heading on the page.

<h1>This is a Heading 1</h1>

<p> - Paragraph

The <p> tag is used for writing paragraphs of text.

<p>This is a paragraph of text.</p>

<ul> and <li> - Unordered List

The <ul> tag creates an unordered list, and the <li> tag is used for each list item.

<ul>
  <li>List Item 1</li>
  <li>List Item 2</li>
  <li>List Item 3</li>
</ul>

<a> - Anchor (Link)

The <a> tag is used to create links. You add a href attribute to tell the browser where to go when the link is clicked.

<a href="https://www.example.com">Visit Example</a>

<img> - Image

The <img> tag is used to display an image on the page. The src attribute specifies the image file.

<img src="image.jpg" alt="Description">

<div> - Division

The <div> tag is used to group content together for styling or layout purposes.

<div class="container">Content goes here</div>

<span> - Span

The <span> tag is used to style a part of text or inline elements within a block.

<span style="color: red">This is red text</span>

<code> - Code

The <code> tag is used to display code inline.

<code>console.log('Hello, world!')</code>

<pre> - Preformatted Text

The <pre> tag is used for preformatted text, where spaces and line breaks are preserved.

<pre>
This is some preformatted text
that preserves spacing and
line breaks.
</pre>

CSS Properties

color

The color property sets the color of the text.

p {
    color: #333;
}

background-color

The background-color property sets the background color of an element.

div {
    background-color: #f0f0f0;
}

font-size

The font-size property changes the size of the text.

h1 {
    font-size: 2rem;
}

padding

The padding property adds space inside an element, between the content and the border.

div {
    padding: 20px;
}

margin

The margin property adds space outside an element, around the content.

p {
    margin: 10px;
}

border

The border property adds a border around an element.

div {
    border: 1px solid #333;
}

font-family

The font-family property sets the typeface for the text.

body {
    font-family: 'Arial', sans-serif;
}

text-align

The text-align property aligns the text within an element.

h1 {
    text-align: center;
}
